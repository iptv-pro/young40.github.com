这两天写一个方块消除游戏, 开局的时候就需要随机颜色方块布局. 在调试的时候, 发现似乎有些关卡总是相同的.
终于发现了程序第一次打开的时候, 开局总是相同的.

<!--more-->

rand()可以产生0到之间的伪随机数, 可以利用srand()指定一个无符号整数作为种子. 如果种子相同, 那么产生的伪随机数列相同.
默认种子为1, 所以如果不改变种子的话, 获得的随机数列总是相同的.

``` cpp Random
srand( time(0) );

for (int i=0; i<10; ++i)
{
    printf("%d\n", rand());
}
```
